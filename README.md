# Restaurant-serve-system

## Project Description
**TableTap** is a web application for restaurant discovery, table booking, and online pre-ordering.  
The platform is designed for two types of users: **clients** and **restaurants**.

Clients can:
- view restaurants on a map;
- see available seats;
- register and log in;
- book tables by selecting date, time, and number of guests;
- browse the online menu;
- add food items to a pre-order;
- leave comments to customize dishes;
- pay for reservation fees on holidays and for pre-orders.

Restaurants can:
- manage their profile and restaurant information;
- update available seats and reservations;
- manage online menus;
- confirm and process customer bookings and pre-orders.

The main goal of the project is to make restaurant booking more convenient by combining **map search**, **table reservation**, **menu pre-ordering**, and **online payment** in one system.

## Main Features
- Restaurant map with available seats
- Registration and authentication for clients and restaurants
- Table reservation by date, time, and number of guests
- Online menu with dish customization through comments
- Reservation and pre-order payment
- User profile with booking history, favorites, and frequent restaurants
- News and promotions page

## Tech Stack
- **Front-end:** Angular
- **Back-end:** Django + Django REST Framework
- **Authentication:** Token/JWT-based authentication
- **Database:** PostgreSQL / SQLite (for development)
